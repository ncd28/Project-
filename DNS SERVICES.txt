**What is DNS
DNS (Domain Name System) is like the phonebook of the internet. It converts human-readable domain names
 (like www.google.com) into IP addresses (like 142.250.183.206) that computers use to communicate.

Record Type	    Purpose	                     Example
A	         Hostname → IPv4 address	     www → 192.168.1.10
AAAA	         Hostname → IPv6 address	
CNAME	         Alias of another name	             ftp → www.example.com
MX	         Mail server	                     mail → 192.168.1.20
PTR	         Reverse DNS	                     192.168.1.10 → printer.local
TXT	         Text info (e.g., SPF, verification)

-------------------------------DNS SERVICES--------------------------------




 1. Name Resolution Service (A, AAAA records)
Domain name → IP address mapping

e.g., client1.biz.local → 192.168.1.10

 2. CNAME Service (Alias Mapping)
One domain points to another

e.g., www.client.biz → server1.client.biz

 3. MX Record Service (Mail Exchange)
Email domain pointing setup

e.g., mail.client.biz → mail server IP

4. PTR Record Service (Reverse Lookup)
IP address → Domain name mapping

 9. DNS Forwarding / Caching
